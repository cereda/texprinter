Released Nov 1st, 2011

Fixed bugs

  1. Potential environment issue when generating TeX.
  2. No support for migrated questions and answers.
  3. No support for community wiki questions and answers.
  4. Some annoyances in the question retrieval algorithm.
  5. Error while trying to fetch a nonexistent question.
  6. Generation of problematic files when an error was raised.

New features

  1. New interface design.
  2. Version checker to inform if there is a newer version available.
  3. Semi-automatic error submission, with report generation.
  4. Better support for migrated questions and answers.
  5. Better support for community wiki questions and answers.
  6. Application changelog description.
  7. Friendly messages instead of cryptic ones.

Improved code

  1. Walktrough execution plan.
  2. Consistent logging API.
  3. Better question retrieval algorithm.
  4. Newer generators.
  5. Better support for command line mode.
  6. New nonexistent question checking method.
  7. Better interfaces.
